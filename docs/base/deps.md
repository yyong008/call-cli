# 依赖

## 描述

本项目基于 `nodejs` 开发，所以需要安装 `nodejs` 环境。当然你如果要贡献代码，需要了解 windows cmd/powershell 命令。以及 linux/macOS 的命令。最后还需要 nodejs 生态完成跨平台的开发。

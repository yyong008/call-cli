# 简介

> icall-cli 目标是在不同的系统提供统一的命令行工具，实现跨平台。为了日常开发中更加翻遍快捷，icall-cli 依赖的很多使用的工具如 `cowsay` 等等。

## 安装

```sh
npm install -g icall-cli
```

## 使用实例

```sh
icall cowsay hi icall ~
```
